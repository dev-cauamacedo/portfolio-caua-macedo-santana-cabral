# 🤖 Análise Crítica do Viés em Modelos de Classificação por IA

[Experiência 02.pdf](https://github.com/user-attachments/files/27560379/Experiencia.02.pdf)

## 📝 Descrição do Projeto

Este projeto consiste na análise prática dos impactos causados pelo viés em modelos de Inteligência Artificial treinados com datasets limitados e estereotipados. Utilizando a plataforma Teachable Machine, foram criadas categorias de classificação visual com base em padrões restritos de imagens, permitindo observar como a ausência de diversidade nos dados compromete a capacidade de generalização do algoritmo.

Durante os testes de inferência, o modelo apresentou classificações incorretas ao analisar indivíduos fora dos padrões utilizados no treinamento, evidenciando limitações técnicas relacionadas à qualidade e representatividade dos dados. O projeto também aborda os impactos sociais gerados por decisões enviesadas da IA, além da importância da curadoria humana (“Human-in-the-loop”) para garantir maior equidade, confiabilidade e responsabilidade no desenvolvimento de sistemas inteligentes.

## 🚀 Tecnologias Utilizadas

* Teachable Machine
* Inteligência Artificial
* Machine Learning
* Classificação de Imagens

## 📊 Resultados e Aprendizados

O projeto permitiu compreender como datasets enviesados afetam diretamente o comportamento de modelos de Inteligência Artificial.

* **Viés Algorítmico:** Identifiquei como padrões limitados de treinamento influenciam decisões incorretas da IA.
* **Generalização:** Observei dificuldades do modelo em reconhecer perfis fora do padrão utilizado no dataset.
* **Impacto Social:** Analisei os riscos sociais e emocionais causados por classificações estereotipadas.
* **Curadoria Humana:** Compreendi a importância da validação humana para reduzir vieses em sistemas inteligentes.

## 🔧 Como Executar

1. Acesse a plataforma Teachable Machine.
2. Crie as categorias de classificação desejadas.
3. Realize o treinamento do modelo com imagens.
4. Execute os testes de inferência utilizando webcam ou imagens externas.
