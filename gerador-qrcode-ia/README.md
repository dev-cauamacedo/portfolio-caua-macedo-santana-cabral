## 📝 Descrição do Projeto

Este projeto consiste na reconstrução de um aplicativo web gerador de QR Code utilizando técnicas de engenharia reversa de interface e desenvolvimento assistido por Inteligência Artificial. O objetivo principal foi reproduzir o comportamento visual e funcional de uma aplicação existente sem acesso ao código-fonte original.

Desenvolvido como parte da disciplina de **Desenvolvimento Assistido por IA**, o sistema foi criado a partir da análise da interface de referência e geração automática de código utilizando o Google AI Studio (Gemini). A aplicação permite gerar QR Codes personalizados através de textos ou links fornecidos pelo usuário, utilizando tecnologias modernas de Front-End.

*Figura 1: Interface principal do sistema de geração de QR Code.*

## 🚀 Tecnologias Utilizadas

* **Framework:** React
* **Linguagem:** TypeScript
* **Estilização:** Tailwind CSS
* **Ferramentas:** Vite, Google AI Studio (Gemini)

## 📊 Resultados e Aprendizados

O projeto permitiu aplicar conceitos modernos de desenvolvimento Front-End e desenvolvimento assistido por IA.

* **Engenharia Reversa:** Aprendi a analisar interfaces e reproduzir funcionalidades sem acesso ao código original.
* **Componentização:** Utilizei React para estruturar componentes reutilizáveis e organizados.
* **Manipulação de Eventos:** Desenvolvi interações dinâmicas utilizando TypeScript.
* **Desenvolvimento Assistido por IA:** Utilizei o Google AI Studio como apoio na geração e estruturação da aplicação.
