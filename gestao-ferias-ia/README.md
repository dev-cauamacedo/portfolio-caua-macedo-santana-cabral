# 🌐 Sistema de Gestão com Bubble.io
 
## 📝 Descrição do Projeto

Este projeto consiste no desenvolvimento de uma aplicação web de gestão utilizando a plataforma Bubble.io com suporte de Inteligência Artificial. O objetivo principal foi criar um sistema funcional aplicando conceitos de modelagem de dados, workflows, segurança e governança em um ambiente Low-Code.

Desenvolvido como parte da disciplina de **Desenvolvimento Assistido por IA e Low-Code**, o sistema foi criado a partir da geração automática de estruturas utilizando IA no Bubble, seguida de ajustes manuais para melhoria da lógica, organização dos fluxos e configuração das permissões de acesso dos usuários.

A aplicação foi estruturada com foco em segurança e escalabilidade, utilizando regras de privacidade, Option Sets para controle de status e organização de workflows para evitar dependência de textos fixos no sistema.

## 🚀 Tecnologias Utilizadas

* **Plataforma:** Bubble.io
* **Metodologia:** Desenvolvimento Low-Code
* **Recursos:** Inteligência Artificial do Bubble
* **Conceitos Aplicados:** Workflows, Privacy Rules, Option Sets e Responsividade
 
## 📊 Resultados e Aprendizados

O projeto permitiu compreender os desafios e limitações do desenvolvimento assistido por IA em plataformas Low-Code.

* **Segurança e Privacidade:** Configurei regras de acesso para impedir a visualização de dados entre diferentes usuários.
* **Modelagem de Dados:** Estruturei entidades e relacionamentos de forma otimizada para escalabilidade.
* **Organização de Workflows:** Corrigi e documentei fluxos gerados automaticamente pela IA.
* **Governança:** Aprendi a utilizar boas práticas de documentação e padronização de processos dentro da plataforma.
* **Desenvolvimento Assistido por IA:** Utilizei IA como apoio para acelerar o desenvolvimento sem depender totalmente da automação.
