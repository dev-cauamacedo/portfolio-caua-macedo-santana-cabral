# 📊 RH Produtivo - Gestão de Projetos e IA

[Experiência 3.pdf](https://github.com/user-attachments/files/27560924/Experiencia.3.pdf)

## 📝 Descrição do Projeto

Este projeto apresenta o desenvolvimento de uma Single Page de Gestão de Projetos e Produtividade para RH, criada através de técnicas avançadas de Engenharia de Prompt.

O objetivo central foi realizar uma “Batalha de Modelos” (LLMs), testando como diferentes inteligências artificiais, como ChatGPT, Claude, Gemini, DeepSeek, Qwen, Grok e Maritaca, interpretam instruções estruturadas em XML.

A aplicação foi desenvolvida com foco em transformar processos burocráticos em fluxos mais ágeis e eficientes, oferecendo ferramentas visuais como Dashboards de People Analytics, Pipeline de Talentos (Kanban) e Planos de Desenvolvimento Individual (PDI).

---

## 🚀 Tecnologias e Metodologias

* **Linguagens:** HTML5 e CSS3
* **Metodologia:** Engenharia de Prompt Estruturada em XML
* **Modelos Testados:** GPT, Gemini, DeepSeek, Qwen, Grok, Maritaca e Claude
* **Conceitos Aplicados:** Layout Responsivo, UX Acadêmica e Single Page Application (SPA)

---

## 📊 Resultados e Aprendizados

O experimento permitiu uma análise aprofundada sobre eficiência, criatividade e comportamento das IAs generativas.

* **Análise de Verbosidade:** Foi identificada uma variação superior a 400% no consumo de tokens entre os modelos para o mesmo resultado técnico.
* **Performance:** O modelo Claude apresentou maior compreensão estética e complexidade de código, enquanto o Gemini destacou-se pela eficiência e menor custo computacional.
* **Fidelidade Estrutural:** Todos os modelos mantiveram 100% de fidelidade à estrutura XML definida no prompt.

---

## 🔧 Como Executar

1. Faça o download dos arquivos do projeto.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Para replicar os testes, utilize o prompt XML disponível na pasta `/prompts` em qualquer uma das IAs utilizadas no experimento.
